# Instalación de react-router-dom

Ejecuta el siguiente comando en tu terminal dentro de la carpeta Frontend:

```bash
cd Frontend
npm install react-router-dom
```

O si prefieres usar la ruta completa:

```bash
cd c:\Users\MattR\Proyecto_A.E.G.I.S\Frontend
npm install react-router-dom
```

Después de instalar, puedes iniciar el servidor de desarrollo con:

```bash
npm run dev
```
